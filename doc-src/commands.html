The Qtile command interface consists of a flat, non-hierarchical namespace of functions. The commandset serves a number of purposes: - Commands can be bound to keys in the Qtile configuration file. - Commands can be called manually through __qsh__, the Qtile shell. - Commands can be called through Qtile's scripting API. This page provides a signature and description for each of the available commands - feel free to experiment using __qsh__! Note that a number of the functions available are only useful when testing and debugging Qtile.
